# Contributing to ConcordOS CE

Thank you for helping advance local-first AI.

Ways to contribute:
- Fix bugs or submit improvements
- Add new DTU tools
- Expand PersonaOS personalities
- Improve CRETI transformations
- Add documentation or tutorials

Before submitting:
- Make sure code runs in `node >= 18`
- Ensure responses remain offline-safe
- Avoid adding telemetry or analytics

PRs welcome.
